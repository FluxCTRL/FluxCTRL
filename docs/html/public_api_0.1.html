post /feeds

Create a new feed

  • Request
  • Response

Body

Type: application/x-www-form-urlencoded

Form Parameters
  • Title: required (string)

    A feed's title

    Example:

    The Oatmeal
  • URI: required (string)

    A feed's endpoint

    Example:

    http://feeds.feedburner.com/oatmealfeed
  • Website: required (string)

    A feed's website

    Example:

    http://theoatmeal.com
  • Status: (boolean)

    A feed's status

    Example:

    true

HTTP status code 200

Everything worked as expected

Body

Type: application/json

Example:

{
  "feed": {
    "id": "1A0iEVP",
    "title": "The Oatmeal",
    "uri": "http://feeds.feedburner.com/oatmealfeed",
    "website": "http://theoatmeal.com",
    "is_active": true,
    "checked": "2015-03-01T00:00:00.000Z"
  }
}

HTTP status code 400

Invalid request parameters

Body

Type: application/json

Example:

{
  "success": false
}
